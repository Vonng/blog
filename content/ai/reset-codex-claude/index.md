前两天，老冯在《[Codex/Claude 脚踏车要蹬冒烟了](/ai/bicycle/)》里说过：**有水快流**。

昨天晚上，老冯刚把三个每月 200 美元的 Claude Code 订阅烧得干干净净。今天早上一睁眼，又迎来特大喜报：Codex 重置了，ChatGPT Work 也重置了。

Claude 当然不甘示弱，[宣布将原定今天结束的 Fable 5 访问延长到 7 月 19 日](https://x.com/claudeai/status/2076351399999557669)。这已经是我们第三次看到 Fable 访问延期了。与此同时，Claude Code 的每周总使用限额继续上调 50%，同样延长到 19 日。

OpenAI 这边更狠。[Codex 与 ChatGPT 负责人 Tibo 宣布](https://x.com/thsottiaux/status/2076365965915467978)：暂时取消 Plus、Business 和 Pro 计划的五小时使用限制，只保留周限额；GPT-5.6 Sol 还会进一步优化效率，让同样的额度跑得更远。更重要的是，Codex 活跃用户已经突破 600 万，于是官方又送了一次额度重置。

照这个增长势头，搞不好明天就要庆祝 700 万了。两边在 X 上针锋相对，用户在下面笑得合不拢嘴——这真是天大的便宜！

## 大撒币与抠抠搜搜

不过，跟 Codex 相比，Claude 还是显得有些抠抠搜搜。

Anthropic 只是说，Fable 5 可以继续用到 19 日，却没有像上次那样直接重置使用额度。老冯的 Fable 额度昨晚已经烧到 100%，现在还得傻等三天，等下一轮额度恢复。

Codex 这边的营销就朴实无华：只要大撒币，大家自然把你捧成圣徒。Claude Code 就显得扭扭捏捏了。老冯坦率地说，我花 200 美元订阅 Claude Max，奔的就是 Fable 5。要是没有 Fable，我可能只会留个 20 或 100 美元的计划，偶尔拿来做一次对抗性 Review。

两边竞争如此激烈，几乎是针锋相对、贴脸开打。当然，最终还是鹬蚌相争，渔翁得利。这也再次用鲜活的案例说明：市场只有引入竞争，用户才能得到实惠；哪一家垄断都不行。

## 两种模型，两种分工

老冯最近用两个 Codex 干了不少大活：收集并校对 PostgreSQL 扩展生态里 1600 多个扩展的详细信息、文档与使用手册；做了一个自用的 APT/YUM 仓库管理工具 SOW；还启动了一个用 Go 重构 Patroni 的项目。

就拿 SOW 来说，我先用 Claude Fable 5 配合 BMAD 做设计、出 PRD，再让 Codex 5.6 Sol Ultra 蹬车。设置好一个 Goal，一干就是 30 多个小时，周额度都打爆了。最后硬是靠着“最后一个任务继续执行”的机制，超额跑完了这轮工作，给我干出一个 2.7 万行代码的初版。今天额度一重置，马上又满血复活，接着蹬车。

就干活而言，Codex 5.6 已经非常令我满意了。我的偏好大致是这样的：

- **日常思考、写文章、创造性设计：** 需要从零开始做设计，或者追求极致智能时，我更愿意使用 Fable。
- **可靠执行、稳定交付：** 需要把明确的设计落成代码时，我会使用 Codex 5.6 Sol Max 或 Ultra。
- **Code Review：** 通常由 Codex 发起，总结上下文并给出第一轮 Review 意见，再自动转交 Fable 5 做对抗性审核。一般两轮达成的共识，就已经相当可靠。

最近我用这套方法把许多旧项目重新扫描了一遍，确实又修出了一批问题。很强大。

## 最后一个任务的额度技巧

这里再分享一个 Codex 小技巧。这不是官方承诺，而是我自己多次观察到的实际现象：**当最后一个任务超出周限额时，系统通常仍会尽最大努力把它执行完。**

举个例子：这周的周限额只剩 1%，此时开启一个足够大的长期任务，系统并不会在额度归零的瞬间立刻掐断，而是可能继续执行很长时间。在极端情况下，最后一个任务几乎还能额外跑掉相当于一整周的额度。

因此，一次 Codex Reset 不只是送你一周配额。如果把最后一个任务安排好，实际效果可能接近两周——差不多相当于半个月的订阅量。

当然，这个机制随时可能调整，也不能把它理解成真正的“无限量”。但按照老冯自己的重度用量折算，一个 200 美元的订阅，每月打满大约能撬动 5000 美元 API 列表价的算力；一次额外重置，相当于白送一份价值近万元人民币的 Token。再叠加最后一个任务的超额执行，单次重置的实际价值可能达到 1.5 万元甚至更高。要是手里有两个号，每次重置都像是捡了两三万元。

## 有水快流，过期不候

总的来说，还是那句话：AI 时代最大的红利之一，就是这些 Coding Plan。

每月 200 美元的订阅，可以撬动约 5000 美元列表价的算力。有水快流，过期不候。在这个关键的时间节点，这些 Token 如果使用得当，可以发挥出巨大的杠杆。

老冯已经准备开第四个 200 美元订阅了。
